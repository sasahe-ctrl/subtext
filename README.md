Stella · 配文搭档

窗外在下雨。那些Ta没有看懂的伏笔，我在这里陪你写完。

一个部署在浏览器里的私人AI对话工具。手机打开即用，不需要安装任何App。

它能做什么
写配文 — 告诉它你拍了什么，光线、地点、心情说一两句，它帮你写出适合发小红书或朋友圈的文字。风格文艺有情绪，不鸡汤，不矫情。
陪你聊 — 也可以什么都不说，就是想说说话。它会在。

使用方式

打开链接，进入时填一下名字，选今天来做什么
描述你看到的画面，或者直接说心情
回车或点发送，也可以点麦克风说话
复制它写的文字，直接发出去

语音输入 — 点左边的麦克风，说完自动发送。支持 Safari 和 Chrome。

技术说明

单文件 HTML，无需后端，无需安装
调用 Claude API（通过 leafapi.cc 转发层）
模型：claude-opus-4-7
支持 GitHub Pages 部署
